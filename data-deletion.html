Exclusão de Dados – MasterPrice

O aplicativo MasterPrice respeita a privacidade dos usuários e oferece meios para a exclusão dos dados associados ao uso do aplicativo.

Como solicitar a exclusão dos dados

O MasterPrice não exige criação de conta pessoal. Os dados estão associados apenas a identificadores anônimos.

  1. Desinstalar o aplicativo do dispositivo;
  2. Limpar os dados do aplicativo nas configurações do sistema;
  3. Redefinir o ID de publicidade nas configurações do Google.

Tipos de dados excluídos

  • Dados de uso do aplicativo;
  • Identificadores anônimos;
  • Histórico local de compras;
  • Registros temporários do app.

Dados que podem ser mantidos

Alguns dados podem ser mantidos de forma anônima e agregada para fins legais, prevenção de fraudes e melhoria do serviço.
